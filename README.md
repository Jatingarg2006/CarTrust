# CarTrust 🚗

CarTrust is a modern web application designed to help first-time used car buyers analyze vehicle condition, maintenance risk, and overall reliability before making a purchase decision.

The project simulates a smart used-car analysis platform using HTML, CSS, and JavaScript with dynamic scoring logic and interactive UI.

---

## ✨ Features

- 🚘 Used car condition analysis
- 📊 Dynamic car scoring system
- 🧠 Intelligent recommendation engine
- 🎨 Modern responsive UI
- 🌙 Premium dark-themed design
- 📱 Mobile responsive layout
- 🔥 Real-time DOM updates
- ✅ Form validation
- 🏷️ Branding with custom logo
